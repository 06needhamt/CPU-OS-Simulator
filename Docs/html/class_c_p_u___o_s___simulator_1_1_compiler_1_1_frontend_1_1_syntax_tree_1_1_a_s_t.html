CPU_OS_Simulator.Compiler.Frontend.SyntaxTree.AST.AST ( Comparison< BaseNode >  theCompareFunction)

The AST constructor requires that we pass a comparison function. We need one as generics can only be compared as equals, but not for order. The solution is to allow the caller to pass a suitable comparison function. We use the C# Comparison delegate for this (found in System.Collections)
